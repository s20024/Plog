# デザインする際に便利なサイトをまとめました

デザイン作業で役立つサイトをカテゴリごとにまとめました。  
ログイン不要、無料で利用できるものを中心に選んでいます。

## 色選択

- [HUE/360](https://hue360.herokuapp.com/)  
  ベースの色を選択し、それに沿った配色を簡単に選べます。
- [ColorSupplyyy](https://old.colorsupplyyy.com/app/)  
  サンプルアイコンや配色の可視化があり、直感的に使いやすいです。

## アイコン

- [Font Awesome](https://fontawesome.com/)  
  豊富なアイコンを簡単に利用できます。

## フォント

- [Google Fonts](https://fonts.google.com/)  
  無料で多くのフォントを利用可能。デザインの定番です。

## CSS

- [UIVerse](https://uiverse.io/)  
  アニメーション付きのデザインが豊富で、コードも確認できます。
- [AnimatiSS](https://xsgames.co/animatiss/)  
  CSSアニメーションを視覚的に確認でき、参考になります。
- [ベイジの図書館](https://baigie.me/officialblog/2021/02/25/css-tips-1/)  
  よく使うCSSコードがまとめられた便利なブログです。
- [外国のおじさんのサイト](https://bennettfeely.com/)  
  CSS生成ツールが充実しています。
  - [Pie chart](https://bennettfeely.com/csspiechart/): 円グラフ作成
  - [Clippy](https://bennettfeely.com/clippy/): 形付き要素作成
  - [Scales](https://bennettfeely.com/scales/): グラデーション作成

## 画像

- [Placehold](https://placehold.jp/)  
  ダミー画像を生成するURLを発行できます。ピクセル数表示も便利です。  
  `https://placehold.jp/300x100.png`  
  ![Sample (300 x 100)](https://placehold.jp/300x100.png)
- [Lorem Picsum](https://picsum.photos/)  
  写真を使ったダミー画像を生成できます。  
  `https://picsum.photos/300/100`  
  ![Sample (300 x 150)](https://picsum.photos/300/100)
